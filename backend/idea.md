# Hospital Appointment System

## Project Overview

The Hospital Appointment System is a web application developed using the MERN Stack technologies. The main purpose of the system is to simplify the process of booking and managing appointments between patients and doctors.

Traditionally, patients need to visit hospitals or make phone calls to book appointments. This system digitizes the process by allowing patients to register, log in, view available doctors, and book appointments online.

The application also provides doctor management features and appointment tracking, making it easier for hospital administrators to organize patient visits.

---

# Problem Statement

Many small hospitals and clinics still rely on manual appointment booking methods. This often leads to:

* Long waiting times
* Appointment conflicts
* Poor record management
* Difficulty tracking patient appointments
* Increased administrative workload

The proposed system addresses these issues by providing an online appointment booking platform.

---

# Objectives

1. Provide secure patient registration and login.
2. Allow administrators to manage doctor information.
3. Enable patients to view doctors and book appointments.
4. Maintain appointment records digitally.
5. Reduce manual paperwork.
6. Improve appointment management efficiency.

# Users of the System

* Register an account

## Doctor

Doctors can:

* View appointment schedules

## Administrator

Administrators can:

* Add doctors
